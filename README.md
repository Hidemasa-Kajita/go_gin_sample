# go_api_sample

## usage
- go api application sample with gin.

## how to use
- execute command
```
$ task build

$ task up
```

- access to browser
    - [http://localhost:8080/](http://localhost:8080/)

## task command
- task build
- task up
- task stop
- task down
- task ps

## packages
- [github.com/gin-gonic/gin v1.7.4](https://github.com/gin-gonic/gin)
